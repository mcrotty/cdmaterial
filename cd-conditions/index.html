Most computer programs are required to make decisions, about what instructions they should do next. The most common one is the `if` statement. This boils down to reasoning as follows: "If this condition is true, do the following .....". This can be illustrated in an example.

var pocket_money = 53.09;
var amount = 39.99;

if (amount < pocket_money) {
   // Condition was true
   console.log("I want to buy this game");
}

In the above example, this will print "I want to buy this game" on the console.

If true do this, otherwise to that

It is very common to have reasoning like: "If this condition is true, do the following ..... Otherwise do these other actions". This is supported by `else` as shown in the following example:

var pocket_money = 53.09;
var amount = 39.99;

if (amount < pocket_money) {
   // Condition was true
   console.log("I want to buy this game");
} else {
   // Condition was false
   console.log("Cant buy it now, keep saving.");
}

All If statements have the following format, where giving the otherwise parts i.e else if (condition2) { ... } or else { ... } are optional. 

if (condition) {
  // Then:
  // Things to do if condition evaluates to true
  // Any amount of statements between { and }
} else if (condition2) {
  // Otherwise check condition2
  // Things to do if condition2 evaluates to true
  // Any amount of statements between { and }
} else {
  // Otherwise:
  // Things to do if condition  and condition2 evaluated to false
 }

Tip

You can add more than one `else if (...)` but you are only allowed one `else`.

You need to be familiar with this, for the next steps.

There is a special operator `%` that can be used to work out the remainder when doing division. This can be shown the following example:

var pocket_money = 53;
var amount = 39;
var remainder = pocket_money % amount;

console.log("If you buy this game, you will have ");
console.log( remainder + " euro left over");
