An element is a pair of HTML tag. Eg. <p>....</p>.

All HTML tags should be paired.

See the `div` element in the example below.

<div class="row">
   <!-- Lots of other tags -->
</div>

In general

Any HTML tag has the following format.

<elementname attributename="something" name2="value2" > 
  Child content
</elementname>

You need to be familiar with this, for the next steps.

Javascript provides some methods to help in adding HTML tags to a web page.

Find the right spot to add the new element.

The following javascript shows how to find a spot in the HTML to add the new element.

var parent = document.getElementById('art');

In this case we have found the <div> with the attribute named "id" is equal to "art"

This spot in the webpage is stored in variable "parent"

Adding the new element.

The following shows how to add an element to the location contained in the variable parent.

parent.appendChild(node);

Where node is a variable containing the element to be added.

The following shows how create a PARAGRAPH <p> element and store it in the variable node.

var node = document.createElement('p');
node.setAttribute("class", "row");

Adding attributes

Remember what we said about HTML tags with attribute names and values. The following code adds a attribute named "mentor" with a value of "micheal".

var node = document.createElement('p');
node.setAttribute("mentor", "micheal");

This would add <p mentor= "micheal"></p> to the web page.

In general

A for loop is a means to do something many times.

for (variable assignment;  condition ; increment) {
  // Something you want to happen many times
}

It stops when the condition is false.

Example

for (var i=0; i < 10 ; i++) {
  // Something you want to happen many times
  console.log("I have done this " + (i+1) + " times");
}
console.log("Ive finished");

Warning

Remember the computer starts counting at zero (0) !
